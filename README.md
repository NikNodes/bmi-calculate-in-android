# BMI Calculator App 📱⚖️

A simple Android application that calculates Body Mass Index (BMI) using height and weight entered by the user.

## Features

- 📏 Input height and weight
- 🧮 Instant BMI calculation
- 📊 Displays BMI result
- 📱 Clean and simple user interface
- ⚡ Fast performance

## Tech Stack

- Java
- Android Studio
- XML
- Android SDK


## How It Works

1. Enter weight
2. Enter height
3. Click Calculate
4. App displays BMI result instantly

## Installation

1. Clone the repository

```bash
git clone <your-repo-link>
```

2. Open in Android Studio

3. Sync Gradle

4. Run the application
